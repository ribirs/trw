 1. 와이어프레임 기본 개념
목적: 사용자가 계정을 등록하기 위한 입력 폼 제공

형태: 모달형 중앙 폼, 백그라운드 어둡게 처리 (Focus 유지)

호출 방식: '회원가입' 버튼 클릭 → 폼 오픈 (모달)

종료 방식: X 아이콘 / 배경 클릭 / ESC 키

 2. 와이어프레임 요소별 구성
위치	요소	설명
상단 중앙	회원가입 버튼	메인 호출 버튼 (고정 위치, 항상 보임)
모달(중앙 정렬)	회원가입 타이틀	<h1>계정 등록</h1>
우측 상단	닫기 아이콘(X)	&times; 버튼으로 모달 종료
입력 영역	폼 필드들	아래 필드들로 구성됨
하단	회원가입 버튼	폼 제출 버튼

3. 입력 필드 구성 순서 (Label + Input)
필드 순서	필드명	타입	설명
1	아이디	text	중복 체크 필요 (현재 랜덤으로 처리)
2	성명	text	5자 제한
3	비밀번호	password	최대 10자
4	비밀번호 확인	password	일치 확인 필요
5	전화번호	tel	기본값 포함
6	이메일	email	기본값 포함
7	주소	text	input[type="text"] 사용
8	생년월일	date	기본값 포함

4. 반응형 설계 요소
Breakpoint	조정 내용
> 768px	라벨과 입력창 수평 정렬, 넓은 입력창
<= 768px	수직 정렬, 입력창 폭 100%
<= 480px	폰트/버튼/여백 축소, 모바일 최적화

 와이어프레임 툴 사용 추천 (선택 사항)
시각적으로 표현하고 싶다면 아래 툴 사용을 추천합니다:

Figma: 무료 와이어프레임 + 프로토타이핑

Wireframe.cc: 간단한 웹 기반 와이어프레임

Balsamiq: 손그림 스타일 와이어프레임

Pen & Paper: 초안이라면 종이 스케치도 좋습니다




1. 전체 구조: 프레임 설계
구성 요소	Figma 요소	설명
전체 화면	Frame	Desktop (1440) 또는 Mobile 프레임 선택
어두운 배경	Rectangle (불투명도 50%)	모달 뒤 배경 (rgba(0,0,0,0.5))
모달 창	Rectangle + Auto Layout	회원가입 입력 박스, 둥근 테두리와 패딩 포함
닫기 아이콘	Text × 또는 Icon	우측 상단, 작은 텍스트 or 아이콘
회원가입 버튼 (열기)	Button (Component)	메인 버튼, 화면 중앙에 위치

2. 모달 내부 구조: Auto Layout 활용
모달 폼을 만들 때 Figma의 Auto Layout 기능을 사용하면 정렬이 쉬워요.

3. 각 필드별 구조
각 필드는 **Auto Layout (Horizontal)**을 사용해 한 줄로 만듭니다.

요소 이름	내용	Figma 설정
Label (텍스트)	아이디, 성명 등	Text size 14~16px, 정렬 왼쪽
Input 필드 (사각형)	직접 사각형으로 제작 후 이름 "Input"	Width: 250~300px, Height: 40px, Border radius: 6px, Fill: #fff
버튼	"회원가입"	Auto Layout 버튼 컴포넌트 or Rectangle + Text, Color: #A5D608

 4. 반응형 고려 팁 (Figma Constraints)
모달 창을 가운데 정렬: Constraints → Center

입력 필드: Width = Fill container로 설정

작은 화면에서는 라벨 위에, 인풋 아래 (수직 정렬) 되도록 Auto Layout → Vertical로 설정

 Figma에서 빠르게 구현하려면:
1. "Desktop Frame" 만들기
Frame Tool (F) → "Desktop (1440x1024)"

2. 버튼 만들기
R로 사각형, T로 텍스트

Shift + A로 Auto Layout 감싸기

Fill: 초록색, Radius: 12px

3. 모달 만들기
사각형 만들고 Shift + A → Auto Layout (Vertical)

패딩 32, 간격 16

내부에 텍스트, 입력 필드 넣기

 텍스트 스타일 가이드 (선택 사항)
유형	크기(px)	폰트 스타일
타이틀 (계정 등록)	24	Bold
라벨	14	Regular
입력 텍스트	16	Regular
버튼 텍스트	16	Bold
